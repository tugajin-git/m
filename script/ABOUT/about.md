# チーム紹介

私たちのチーム「ADRoboRacing」は、自動運転技術の最前線でレース開発を行っています。  
多様なバックグラウンドを持つメンバーが集まり、最先端のAI技術と自動車制御システムを融合し、競技に挑戦しています。
